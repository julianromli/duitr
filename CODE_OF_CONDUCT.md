# Code of Conduct

## Our Standard

We want Duitr to be a respectful, constructive, and welcoming project.

Examples of behavior that support a positive environment:

- Being respectful in discussions and reviews
- Assuming good intent while giving clear feedback
- Accepting constructive criticism
- Focusing on what is best for the project and community

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Discriminatory language or conduct
- Trolling or bad-faith participation
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers through GitHub with enough context for review.
